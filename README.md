# MedSearch - محرك البحث الطبي العالمي

![MedSearch](images/og-image.png)

## 📚 الوصف
محرك بحث طبي عالمي مع قواميس شاملة للأدوية والأمراض والأنسجة والمختبرات، مع ربط مباشر بالمصادر الطبية العالمية.

## 🚀 النشر السريع
1. ارفع جميع الملفات لأي استضافة
2. تأكد من وجود مجلد `images/` مع الأيقونات
3. الموقع جاهز للعمل مباشرة

## 📱 تثبيت كتطبيق
- **أندرويد**: افتح في Chrome ← تثبيت التطبيق
- **iOS**: افتح في Safari ← مشاركة ← إضافة للشاشة الرئيسية
- **كمبيوتر**: Chrome ← أيقونة التثبيت في شريط العنوان
